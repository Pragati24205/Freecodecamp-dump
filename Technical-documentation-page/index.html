Variables and Data Types

Variables in Python are created by assignment. You don't need to declare the type explicitly.

Common data types include int, float, str, bool, list, tuple, dict, and set.

Primitive data type are as follows:

Type Example Description
int x = 5 Whole numbers
float pi = 3.14 Decimal numbers
str name = "hehe" Text (sequence of characters)
bool a = True Boolean (True or False)
NoneType x = None Represents "nothing" or no value

Example Code:

x = 10
name = "hehe"
is_student = True

Type checking and conversion:

type(x) # Returns data type
int("5") # Converts string to int
str(10) # Converts int to string
float("3.14") # String to float

Control Flow

1. Conditional Statements (if, elif, else)

➤Used to make decisions

x = 10
if x > 0:
  print("Positive")
elif x == 0:
  print("Zero")
else:
  print("Negative")

2. Loops

➤ for loop – Iterate over a sequence

for i in range(5):
  print(i) # 0 to 4

➤ You can also loop through strings, lists, etc.

for char in "hehe":
  print(char)

3. Loop control statements

➤ break – Exit the loop early

for i in range(5):
  if i == 3:
    break
  print(i)
# Output: 0 1 2

➤ continue – Skip current iteration

for i in range(5):
  if i == 2:
     continue
  print(i)
# Output: 0 1 3 4

➤ pass – Do nothing (placeholder)

for i in range(5):
   if i == 2:
     pass # placeholder
   print(i)

Python specific tip

  • Indentation is everything in control flow.
  • No {} like in C/C++/Java — you must indent blocks correctly.

Functions

What is a Function?
A function is a reusable block of code that performs a specific task.

1. Defining a Function:

def greet():
  print("Hello!")

2. Calling a Function:

greet()
# Output: Hello!

3. Function with Parameters:

def greet(name):
   print(f"Hello, {name}!")
greet("hehe")
# Output: Hello, hehe!

4. Function with Return Value:

def add(a, b):
   return a + b
result = add(3, 5)
print(result) # Output: 8

5. Default Parameters:

def greet(name="Guest"):
   print(f"Hello, {name}!")
greet() # Hello, Guest
greet("hehe") # Hello, hehe

6. Keyword Arguments:

def info(name, age):
   print(f"{name} is {age} years old.")
info(age=120, name="hehe")

7. Variable-Length Arguments:

➤ *args → Multiple positional arguments

def total(*nums):
   return sum(nums)
print(total(1, 2, 3)) # Output: 6

➤ **kwargs → Multiple keyword arguments

def profile(**data):
   for key, value in data.items():
      print(f"{key}: {value}")
profile(name="hehe", age=120)

Lists and Dictionaries

Lists in Python:

What is a list?

  • Ordered, changeable, indexed
  • Can store mixed data types
fruits = ["apple", "banana", "mango"]

Common list operations:

Operation Example Description
Access fruits[0] Get item by index
Modify fruits[1] = "grape" Change item
Add (end) fruits.append("orange") Add to end
Add (specific index) fruits.insert(1, "kiwi") Insert at index
Remove by value fruits.remove("apple") Remove first match
Remove by index fruits.pop(0) Remove and return item
Length len(fruits) Number of items
Check existence "apple" in fruits Returns True/False
Sort fruits.sort() Sort alphabetically
Reverse fruits.reverse() Reverse order

Dictionaries in python:

What is a dictionary?

  • Unordered (until Python 3.7+ keeps insertion order)
  • Stores key-value pairs
student = { "name": "hehe", "age": 120, "branch": "CSE" }

Common Dictionary Operations:

Operation Example Description
Access value student["name"] Get value by key
Add/update student["age"] = 21 Add or update value
Remove key student.pop("branch") Remove key-value pair
Keys list student.keys() Get all keys
Values list student.values() Get all values
Key-value pairs student.items() Get all pairs
Check key existence "name" in student Returns True/False
Length len(student) Number of key-value pairs
